# Linux文件IO学习计划（C++）

## 学习目标
掌握Linux环境下的文件IO编程技能，能够进行实际应用开发

---

## 阶段划分

### 阶段1：基础概念与开发环境
**目标**：理解Linux文件IO核心概念，搭建开发环境

#### 1.1 核心概念
- 文件描述符（File Descriptor）概念
- 系统调用与库函数区别
- Linux文件系统架构概述
- 路径与工作目录

#### 1.2 开发环境
- GCC/G++编译器配置
- Makefile基础
- GDB调试入门

---

### 阶段2：基础文件操作
**目标**：掌握文件打开、关闭、读取、写入

#### 2.1 核心系统调用
| 函数 | 作用 |
|------|------|
| `open()` | 打开/创建文件 |
| `close()` | 关闭文件 |
| `read()` | 读取数据 |
| `write()` | 写入数据 |

#### 2.2 标志位与权限
- `O_RDONLY`, `O_WRONLY`, `O_RDWR`
- `O_CREAT`, `O_APPEND`, `O_TRUNC`, `O_NONBLOCK`
- 文件权限位（mode_t）

#### 2.3 实践项目
- 简单的文件复制程序

---

### 阶段3：文件定位与状态
**目标**：掌握文件指针操作和文件状态获取

#### 3.1 文件定位
| 函数 | 作用 |
|------|------|
| `lseek()` | 移动文件指针 |
| `ftell()` | 获取文件大小 |
| `fseek()` | C库定位 |

#### 3.2 文件状态
| 函数 | 作用 |
|------|------|
| `fstat()` | 获取文件状态 |
| `stat()` | 路径文件状态 |
| `access()` | 检查文件权限 |

#### 3.3 实践项目
- 文件内容查看器
- 文件大小统计工具

---

### 阶段4：目录操作
**目标**：掌握目录遍历和文件系统操作

#### 4.1 目录操作
| 函数 | 作用 |
|------|------|
| `mkdir()` | 创建目录 |
| `rmdir()` | 删除空目录 |
| `opendir()` | 打开目录 |
| `readdir()` | 读取目录项 |
| `closedir()` | 关闭目录 |

#### 4.2 目录遍历
- 递归遍历目录树
- 过滤文件类型
- 符号链接处理

#### 4.3 实践项目
- 目录树浏览工具
- 文件搜索程序

---

### 阶段5：高级文件操作
**目标**：掌握高级文件特性

#### 5.1 文件锁
- 互斥锁 `flock()`
- 记录锁 `fcntl()`
- 死锁避免策略

#### 5.2 内存映射
- `mmap()` 映射文件到内存
- `munmap()` 解除映射
- 共享内存应用

#### 5.3 特殊文件
- 管道（Pipe）
- 套接字（Socket）基础
- 设备文件概念

#### 5.4 文件属性
- 修改文件时间戳
- 重命名与链接
- 软硬链接区别

#### 5.5 实践项目
- 文件锁实现并发访问控制
- 大文件高效读取（内存映射）

---

### 阶段6：实战项目
**目标**：综合应用所学知识

#### 项目1：配置文件管理器
- 读取/解析配置文件
- 写回配置
- 备份机制

#### 项目2：简单日志系统
- 按日期创建日志文件
- 日志轮转
- 多进程写入文件锁

#### 项目3：文件同步备份工具
- 目录差异检测
- 增量备份
- 进度显示

## 学习建议

1. **循序渐进**：每个阶段都要动手实践，不要只看不动手
2. **查阅手册**：习惯使用`man`命令查阅API
3. **调试能力**：学会使用GDB调试程序
4. **错误处理**：所有系统调用都要检查返回值
5. **总结归纳**：每学完一个模块，整理知识点
